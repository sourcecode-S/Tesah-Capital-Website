## Frontend Framework & Technologies

**Core Framework:**

- **Next.js 14** with App Router - React-based full-stack framework
- **React 18** - Component-based UI library
- **TypeScript** - Type-safe JavaScript


**Styling & UI:**

- **Tailwind CSS** - Utility-first CSS framework
- **shadcn/ui** - Pre-built component library built on Radix UI
- **Radix UI** - Headless UI primitives for accessibility
- **Lucide React** - Icon library
- **class-variance-authority (CVA)** - For component variants


## Backend & Data Management

**Server-Side:**

- **Next.js Server Actions** - For form handling and server-side logic
- **Next.js API Routes** - For backend endpoints


**Data Storage:**

- Mock data structures (no database currently connected)
- Local state management with React hooks


## Development Tools & Build

**Package Manager:**

- **Bun** - Fast JavaScript runtime and package manager


**Deployment:**

- **Vercel** - Hosting and deployment platform (based on error messages)


## Key Features Implemented

**Content Management:**

- Custom CMS-like functionality for managing pages

- Media management system
- Dynamic page builder


**Business Features:**

- Job application system with file uploads
- Investment calculators

**Admin Panel:**

- Dashboard with analytics

## Architecture Highlights

- **Component-based architecture** with reusable UI components
- **Server-side rendering (SSR)** for better SEO and performance
- **Type-safe development** with TypeScript throughout
- **Responsive design** with mobile-first approach
- **Accessibility-focused** using Radix UI primitives


This is a professional, enterprise-grade website built with modern web technologies, focusing on performance, accessibility, and maintainability. The stack chosen is well-suited for a financial services company like Tesah Capital, providing both the functionality needed for business operations and the professional appearance expected in the finance industry
